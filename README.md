# PromptStack 📚

**Prompt version control for AI power users.**

Save, version, compare, and optimize your prompts. Never lose a great prompt again.

## Why PromptStack?

- **Version Control**: Track every iteration of your prompts
- **Compare Versions**: Diff between versions to see what changed
- **Rate Performance**: Star ratings to track which prompts work best
- **Search**: Find prompts by name, content, or category
- **CLI-first**: Fast terminal workflow for power users

## Installation

```bash
npm install -g promptstack
```

## Quick Start

```bash
# Create a new prompt
promptstack create "code-review" -d "Reviews code for bugs" -c coding -t "Review this code for bugs and improvements:"

# List all prompts
promptstack list

# Show current version
promptstack show <prompt-id>

# Add a new version
promptstack version <prompt-id> -t "Improved prompt content" -n "Added examples"

# Compare versions
promptstack diff <prompt-id> 1 2

# Rate a version
promptstack rate <prompt-id> 2 5
```

## Features

### 📝 Prompt Management
- Create prompts with name, description, and category
- Add unlimited versions to each prompt
- Store metadata (model, temperature, notes)

### 🔄 Version Control
- Every change is saved as a new version
- Compare any two versions with diff
- Track performance ratings

### 📊 Analytics
- View stats across all prompts
- See version history
- Track ratings over time

### 🔍 Search
- Full-text search across prompts
- Filter by category
- Quick navigation

## Data Storage

Prompts are stored locally in `~/.promptstack/prompts.json` as a simple JSON file. Easy to backup, export, or version control yourself.

## Use Cases

- **Prompt Engineers**: Iterate on prompts systematically
- **AI Researchers**: Track experiments and results
- **Developers**: Version control prompts alongside code
- **Content Creators**: Maintain a library of effective prompts

## API

PromptStack can also be used as a library:

```typescript
import { PromptStack } from 'promptstack';

const ps = new PromptStack();

// Create a prompt
const prompt = ps.create('my-prompt', 'Description', 'category', 'Prompt content');

// Add versions
ps.addVersion(prompt.id, 'Improved content', { notes: 'Added examples' });

// Get current version
const current = ps.getCurrent(prompt.id);

// Compare versions
const diff = ps.compare(prompt.id, 1, 2);
```
